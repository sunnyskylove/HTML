폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다.
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다.
* action 속성 : 폼의 입력된 값들을 전송받을 서버의 URL 을 입력한다. * method 속성 : get/post 방식으로 전송 방식을 지정한다. - get : url 주소에 form 태그를 통해 넘어가는 데이터가 보임.(url정보노출) - post : url 주소에 form 태그를 통해 넘어가는 데이터가 보이지 않음. (url 정보 노출안됨)

submit 버튼을 눌렀을때 action 에 지정한 경로로 methoed에 지정한 방식에 따라
input 태그에 입력한 값들을 전달한다.
